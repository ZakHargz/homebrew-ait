# Zakhargz Ait

## How do I install these formulae?

`brew install zakhargz/ait/<formula>`

Or `brew tap zakhargz/ait` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "zakhargz/ait"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
